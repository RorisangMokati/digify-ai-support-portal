# digify-ai-support-portal
Frontend and AI support operations portal project for Digify CX

## Project Purpose

This project aims to build a working AI-powered support operations portal that:

* Captures support/project requests
* Validates user input
* Connects to backend APIs
* Tracks requests and reporting

## Tech Stack

* React
* GitHub
* Netlify
* Cloudflare Workers
* Groq AI

## Current Status

Project setup and research phase.
